## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Rishabh1711-dev/PortFolio.git
```

Navigate into the project

```bash
cd PortFolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```
